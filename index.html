Solar Center

Spring 2024 · Creative Algorithms

Description

SOLAR CENTER is a simple and personalizable meditation guide that aims to help balance hectic lifestyles and mindfulness through breathwork and exercising creativity.

Motivation

As life grew busier than before Sophomore year, balancing so many responsibilities led me to forget the importance in self care and taking breaks. I wanted to create a design/visual/process that could focus my attention at something almost hypnotizing to relax hyper minds. It acts as a brain break, still activating our brains but with less cognitive load.

Inspiration

Seeing an 80s design of a pixelated horizon amidst finals season, I felt there was quite peaceful tone to the simplicity of the graphic. Although it is blocky and game-like, I thought the style could be an interesting take on a meditation guide, contrasting yet oddly coinciding with the calming aspects of meditation.

Design & Implementation

The app was built in Java using the Processing library. It features a dynamic horizon that expands and contracts with the user's breath, a timer for meditation sessions, and customizable color schemes. I focused on creating smooth animations and an intuitive interface to enhance the user experience. Building this project was a rewarding exploration of creative coding and personal wellness. Features I implemented include the breathing horizon animation, session timer, and color customization options.

Timely App

Spring 2025 · Object Oriented Programming

Description

Timely is an interactive app for organizing events and tasks in a planner. It provides a simple calendar view for all-day events and time blocks, plus a unique progress chart that visualizes your task completion in real time.

Motivation

Time management is a skill I've been working on to balance personal and school life. After trying multiple planners, I couldn't find one with everything I needed — so I built my own.

Design & Implementation

The back-end follows the MVC pattern with Observer and Facade patterns. I implemented the To Do List and Progress Chart components, and did a full code cleanup to ensure all lines were relevant. Building interactive UI with MVC was a rewarding learning experience.
